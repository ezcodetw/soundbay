A) FAQ - top

  • First of all, you've to add the the slider's content (image, text etc) in the HTML, which looks like:
    	<div id="banner">  				
		<div class="oneByOne_item">
			<span class="text1">iOS Friendly</span>			
			<span class="text2">Drag and throw the banner</span>
			<img src="images/books/book1.png" class="img1" />
			<img src="images/books/book2.png" class="img2" />
		</div>
		<div class="oneByOne_item">                          
			...
		</div>
		<div class="oneByOne_item">
			...
		</div>
		<div class="oneByOne_item">
			...
		</div>                                                                                              
		<div class="oneByOne_item">                                 	
		    ...
		</div>			
	</div>
    Then use our jQuery plugin to generate the slider: 
     
   $(document).ready(function() { 
       $('#banner').oneByOne({
		className: 'oneByOne1',	             
		easeType: 'random'
	});  
   });
    The #banner is the slider main container of the slider in the HTML. The oneByOne1 is the generated by the plugin which wrap the #banner. Then customize the css style of each elements in the HTML.
    The whole available settings of the plugins: 
     
   $(document).ready(function() { 
       $('#banner').oneByOne({
			className: 'oneByOne', // the wrapper's name
			sliderClassName: 'oneByOne_item',  // the each slider div's class name
			easeType: 'fadeInLeft',  // the ease animation style
			width: 960,  // width of the slider
			height: 420, // height of the slider
			delay: 300,  // the delay of the touch/drag tween
			tolerance: 0.25, // the tolerance of the touch/drag  
			enableDrag: true,  // enable or disable the drag function by mouse
			showArrow: true,  // display the previous/next arrow or not
			showButton: true,  // display the circle buttons or not
			slideShow: false,  // auto play the slider or not
			slideShowDelay: 3000 // the delay millisecond of the slidershow
	});  
   });
  • How many ease animation style does this plugin support?
    We use CSS3(animate.css) to toggle the animation here, so the easeType can be 'fadeIn', 'fadeInUp', 'fadeInDown', 'fadeInLeft', 'fadeInRight', 'fadeInRight', 'bounceIn', 'bounceInDown', 'bounceInUp', 'bounceInLeft', 'bounceInRight', 'rotateIn', 'rotateInDownLeft', 'rotateInDownRight', 'rotateInUpLeft', 'rotateInUpRight' or 'random'.
    The animate.css is a little big now. You can use the easeType only css to minify the size. Take the example2.html for example. We use the fadeInLeft only. So we can customize the fadeInLeft CSS only in the example2.css. You can find some more information from: http://daneden.me/animate/, and there is a custom build tool there.

B) The source and deploy Folder - top

You only have to upload the files in the deploy folder. The deploy folder contains the minified javascript. You can just update the image content in the index.html by your own ones.

Please don't upload the files in the source folder. The source folder contains the original javascript source, so please don't upload the javascript files in this folder. Thanks.
